This repository was created for studying.

It concerns the web applications I've developed.

The folder called "coursework" contains a *pharmacy website*.
This project is an online multipage pharmacy shop, that implements paging, caching data, injection protection, users’ authorization, passwords’ hashing, AJAX updating, drugs searching and etc.

The folder called "coursework_startup" contains a *website for residents of apartment buildings*.
This application allows residents of the same building to send statements to administrators (managers of the building), to poll, to receive notifications, to pay for services, to check the problems statuses.

The folder called "Software Engineering Components" contains a front-end for the *public associations register website*.
This application implements a web database of Ukrainian public associations with CRUD and searching.
